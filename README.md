# 找回遺失的自己：從忙亂到安靜的力量
## 小學生的關懷倫理學課程設計

本專案為「114-1 情意與道德課程研究」之課程設計成果。旨在透過關懷倫理學 (Ethics of Care) 的視角，引導國小高年級學生在忙碌與混亂中找回自我價值，並學習如何在照顧他人之前，先安頓好自己。

### 📂 專案內容
*   **核心理論**：Noddings 關懷倫理學、情緒覺察 (SEL)。
*   **教學目標**：自我覺察、社會覺察、負責任的決定。
*   **主要活動**：身教故事、道德兩難抉擇、情緒錨定、樹洞對話。
